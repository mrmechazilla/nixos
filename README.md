# NixOS

My personal reproducible **NixOS** configuration for my laptop.

**NixOS** is a Linux distribution based on the Nix package manager, where the entire system (packages, services, users, settings) is defined declaratively in configuration files, making it **reproducible**, **rollback-friendly**, and easy to maintain.

---


## What’s inside

- **Desktop:** GNOME (GDM + Wayland)
- **GPU:** NVIDIA (PRIME offload)
- **Audio:** PipeWire (PulseAudio compatibility)
- **Networking:** NetworkManager
- **Virtualization:** Docker (+ docker-compose)
- **Apps:** Brave, Chrome, Spotify, Obsidian, VS Code, IntelliJ, etc.
- **Flatpak:** enabled + Flathub auto setup after login

---

## Repository structure

- `configuration.nix` → Main system configuration
- `hardware-configuration.nix` → Hardware-specific config (generated by NixOS)

---

## Apply configuration on this machine

From `/etc/nixos`:

### Safe apply (temporary)
```bash
sudo nixos-rebuild test
```

### Permanent apply
```bash
sudo nixos-rebuild switch
```
## Useful commands

### Show system generations
```bash
sudo nix-env --list-generations --profile /nix/var/nix/profiles/system
```

### Roll back to previous generation
```bash
sudo nixos-rebuild switch --rollback
```

### Show last boot errors
```bash
journalctl -b -p err --no-pager
```
